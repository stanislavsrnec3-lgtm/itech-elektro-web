# iTECH elektro web

Jednoduchý firemní web vytvořený jako statický projekt v `HTML + CSS + JavaScript`.

## Co je hotové

- úvodní stránka s výrazným blokem referencí z NejŘemeslníci.cz
- rozcestník pro 3 cílové skupiny
- sekce pro domácnosti, města a úřady, stavební firmy
- sekce dokumentů a certifikací
- kontaktní formulář
- připravené složky pro budoucí fotky a PDF dokumenty

## Jak web otevřít ve Windows

1. Otevřete složku `C:\Users\stani\Desktop\WEB1`.
2. Najděte soubor `index.html`.
3. Dvakrát na něj klikněte levým tlačítkem myši.
4. Web se otevře ve vašem výchozím prohlížeči.

Pokud by se po dvojkliku nic nestalo:

1. Klikněte na `index.html` pravým tlačítkem myši.
2. Zvolte `Otevřít v programu`.
3. Vyberte například `Google Chrome`, `Microsoft Edge` nebo jiný prohlížeč.

## Jak později měnit obsah

- hlavní texty jsou v souboru `index.html`
- vzhled je v souboru `styles.css`
- galerie, veřejné zakázky a dokumenty jsou v souboru `script.js`
- fotky patří do složky `assets/gallery`
- PDF dokumenty patří do složky `assets/documents`

## Co upravíte nejčastěji

- telefon a kontakty v `index.html`
- texty sekcí v `index.html`
- seznam fotek v poli `galleryItems` v `script.js`
- seznam dokumentů v poli `documentItems` v `script.js`
